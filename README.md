# NanaBot
> a Discord/Gitter bot that does silly things

### .env
```
# https://developer.gitter.im/apps
GITTER_TOKEN=?
GITTER_ROOM=56e7130185d51f252ab8c2fa

# https://discordapp.com/developers/applications/me
DISCORD_TOKEN=?
```

### Install
```sh
$ npm install
```

### Run
```sh
$ node bleep-bloop.js
$ node bleep-bloop.discord.js
```
